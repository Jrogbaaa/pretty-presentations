## How It Works

### Template Selection Logic
```
1. Check for brand name match ‚Üí Use matched template
2. Check for technical audience ‚Üí Use Look After You Standard (lower passion)
3. Analyze campaign type ‚Üí Event/Sports = Red Bull, Fashion = Scalpers
4. Evaluate budget + audience ‚Üí Premium fashion = Scalpers
5. Default ‚Üí Look After You Standard
```

### Data Visualization Logic
```
Comparing discrete values (2-10 items) ‚Üí Bar Chart
Showing proportions (2-5 segments) ‚Üí Donut Chart
Showing trends (4+ points) ‚Üí Line Chart
Highlighting single metric ‚Üí Enhanced Metric Card
Making numbers relatable ‚Üí Pictograph + Animated Number
```

### Storytelling Integration
```
Every slide now follows narrative structure:
- Cover: Anticipation (Hook)
- Objective: Purpose (Why)
- Creative: Excitement (Vision)
- Talent: Confidence (Who + Data)
- Scenario: Value (Solution + Charts)
- Next Steps: Optimism (Forward path)
```
